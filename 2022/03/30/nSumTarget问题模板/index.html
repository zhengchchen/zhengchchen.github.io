nSumTarget问题模板

发表于 2022-03-30 | 分类于 leetcode 双指针 |
字数统计: 559 | 阅读时长 ≈ 2

前言

nSumTarget问题为leetcode中常见的求数组中某几个数之和定于固定值的问题。一般形式为给定数组nums，求数组中n个数之和等于target，返回这n个数的索引、值、和、差值等情况。笔者总结了nSumTarget模板，可以解决求任意n个数等于(接近于)任意target的问题。

适用场景

求数组内n个数之和等于(最接近于)target的数，或求数组内n个数之和最小值(最大值)。

基本思路

排序+双指针+递归调用，先对数组进行升序排序，然后将n个数求和问题转化为2个数求和问题，用双指针解决两个数求和问题。

注意点

  1. 跳过所有重复的元素： while (lo < hi && nums[lo] == left) lo++；while (lo < hi && nums[hi] == right) hi--;(在找到一个结果后，也应该跳过所有重复项)

  2. 在twoSum子问题解决后，需要将当前的元素也push进去，即tuple.push_back(nums[i]);

  3. 在一个结果找到后，需要跳过和当前数字重复的情况，否则会出现重复结果;while (i < sz - 1 && nums[i] == nums[i + 1]) i++;

  4. 记得每次调用从start开始，不是0开始，否则会出现重复。
