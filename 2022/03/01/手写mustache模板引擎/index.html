手写mustache模板引擎

发表于 2022-03-01 | 分类于 Vue

mustache模板引擎的作用是将字符串模板变为dom模板，最后结合数据挂载到dom树上，在页面上进行渲染。在这个过程中，mustache将模板字符串先转换为tokens，然后结合数据data，最终生成dom字符串。例如：

实现Scanner类

Scanner类即一个扫描器，用来扫描模板字符串，有两个属性和三个方法：

  • pos：记录字符串当前正被扫描的位置

  • tail：记录当前指针及当前指针之后的位置的所有字符串

  • scanUntil(stopTag)：扫描模板字符串，直到遇见第一个stopTag结束，返回扫描过的字符串，此时调用scan()跳过stopTag

  • scan()：让指针跳过stopTag

  • eos()：判断当前字符串是否扫描完毕

实现ParseTemplateToTokens(templeteStr)函数

ParseTemplateToTokens的作用是将模板字符串变为tokens，过程中用到了上述的Scanner类和nestToken类（后文介绍），有三个属性：

  • tokens：存储tokens结果

  • scanner：用于扫描字符串，跳过结束关键字

  • words：用来保存scaner类扫描过的文字，在合适的地方将文字存入tokens数组

实现nestTokens()函数

如上节所诉，在遇到#字符时，需要进行嵌套存储。nestTokens类用于将一个未嵌套过的tokens处理成嵌套后的nestedTokens返回。

这里，我们需要将#和/之间的字符串放入到当前token数组的第三项，作为它的嵌套项(tokens数组包含若干个token对象，每个token对象又包含类型和内容两个属性，如果当前token又嵌套内容，则单独开辟第三项用于存储)

实现嵌套循环：引入栈和收集器(collector)，一开始，让collector指向nestedTokens结果数组，它是一个引用类型值，遇见#时，将当前的token 放入栈中，并给token添加下标为2的项，并且让collector指向它：collector = token[2] = [],遇见/时，先将栈顶元素退栈，此时改变collector为栈顶元素下标为2的数组，如果栈顶此时无元素，则证明当前的嵌套循环以及完成闭合或者无嵌套，可以让collector重新指向结果数组 collector = sections.length > 0 ? sections[sections.length - 1][2] : nestedTokens;

实现renderTemplate(tokens, data)函数

此时我们已经利用ParseTemplateToTokens(templeteStr)函数得到了tokens数组，现在需要结合数据data将tokens对应部分填充，最终得到dom字符串。

renderTemplate(tokens, data)函数主要通过判断当前token的类型作出相应的处理，用到了两个函数：

  • lookup()：功能是将一个多层嵌套的对象属性剥离开，返回其基础数据类型(例如，n.m.p=123，将’.’去除，最终返回123)；
  • parseArray()：功能是将’#’和’/‘之前的嵌套token展开并返回

lookup()函数的实现：将当前字符串中的’.’去除并保存新字符串中，依次向下获取，直到返回基础数据类型

parseArray()函数的实现：这函数要递归调用renderTemplate()函数，调用的次数由data决定

以上就是整个mustache模板引擎的简单实现，主要步骤在于利用ParseTemplateToTokens(templeteStr)函数将模板字符串转换为tokens，再结合data最后返回dom字符串。
